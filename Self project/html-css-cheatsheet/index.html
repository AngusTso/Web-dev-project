CSS Attributes

CSS Position attributes

Attributes Name Description
Static Static Position (default value) Item stay with the flow of HTML in its default position
Relative Relative Position Position an element relative to its default static position on the web page
Absoulte Absoulte Position The element will be positioned relative to its closest positioned parent element
Fixed Fixed Position Fix an element to a specific position on the page (regardless of user scrolling)
Sticky Sticky Position Keeps an element in the document flow as the user scrolls, but sticks to a specified position

CSS Display attributes

Attributes Default tag Description
Inline span , strong , em A box that wraps tightly around their content, only taking up the amount of space necessary
Block div , p Block level elements are not displayed in the same line as the content around them
Inline-Block N/A Combines features of both inline and block elements
